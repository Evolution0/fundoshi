Fundoshi - The universal manga data extractor
=============================================

Fundoshi is nifty library that lets you search for and extract manga series'
data from many online manga reader websites.

.. doctest::

    >>> import fundoshi
    >>> 1 + 1
    2
    >>> fundoshi
    None
